# PCF-Explorer
A tool to explore PCF dependencies.<br/>

- Type (Field/Dataset)
- Properties (Required, type, name)
- Resources (js/resx)
- Connected Forms/dashboard/Entities
- Solutions
- Search PCF controls
- Quick count of Control Types (Pire Chart)
- Quick glamnce of Control Types
